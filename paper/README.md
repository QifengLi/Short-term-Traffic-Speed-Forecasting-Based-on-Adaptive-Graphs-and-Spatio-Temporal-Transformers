# Final Report Materials

This directory contains the final report document and the lightweight evidence files used to support the reported experimental results.

## Contents

```text
final_report/assets/evidence/   JSON result summaries and significance-test outputs
final_report/assets/tables/     CSV files used as table evidence
final_report/assets/figures/    Figures included in the report
```

The raw datasets, trained checkpoints and full prediction arrays are not included because they are large local artefacts. The included evidence files are intended to make the reported numerical results inspectable without storing the full experiment output directory.
